# 🗺️ Tourism Planner — Reasonix Skill

高质量旅行规划 AI Skill，运行于 [Reasonix](https://reasonix.ai) Agent 框架。根据用户需求生成**真实、可行、可调节**的行程方案，支持多轮对话修改、预算追踪和冲突检测。

## ✨ 核心能力

| 模块 | 说明 |
|------|------|
| **需求理解** | 结构化收集目的地/日期/预算/偏好，**每次最多 3 个问题**，不打扰 |
| **实时数据** | 通过 Tavily / Web Search / Web Fetch 查询航班、酒店、景点、天气、汇率、签证，**绝不编造数据** |
| **行程编排** | 地理顺路 + 时间窗约束 + 用餐缓冲，4 条冲突检测规则自动预警 |
| **结构化输出** | 内部 JSON → 时间轴文本，含地图链接、花费明细、数据来源 |
| **多轮修改** | 按 item `id` 精准修改，支持替换/增删/对调/预算调整 |
| **预算追踪** | 每日小计 + 全程总计，自动对比用户预算范围 |
| **状态持久化** | 每次回复末尾输出 JSON 快照，跨轮对话无缝衔接 |

## 🚀 快速使用

在 Reasonix 中加载此 Skill 后：

```
/tourism-planner
```

按 Skill 引导开场，逐步提供目的地、日期、偏好即可生成完整行程。示例对话：

> 👋 您好！我是您的旅行规划助手。首先，请告诉我：您想去哪里旅行？

> 我想去京都看红叶，11月下旬，5天，两人，预算1.5万以内

Skill 会自动调用搜索工具查询航班、酒店、景点，生成带时间轴和花费明细的行程，并在末尾附上出行提示（签证、电源、礼仪、紧急电话等）。

## 🔧 安装到 Reasonix

1. 将本仓库克隆到本地
2. 在 Reasonix 中使用 `/install-skill` 或 `install_skill` 工具指向 `.reasonix/skills/tourism-planner/` 目录
3. 加载后即可通过 `/tourism-planner` 激活

## ⚠️ 安全与信任铁律

- **绝不**声称已"预订"或"购买"任何东西
- **绝不**编造酒店名称、航班号、价格、开放时间
- 所有价格注明"以预订时显示为准"
- 签证政策注明"请以官方渠道最终确认为准"
